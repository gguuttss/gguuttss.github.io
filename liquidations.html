Liquidations

Handle liquidations to receive rewards. To learn more about the specifics, read the relevant docs section!
Or, for a short overview of the liquidation process, click the button below.

1. Marking a loan for liquidation (left column)
Before a loan can be liquidated, it needs to be marked. The marker of a loan receives a marker receipt. You cannot choose what loan to mark, the loan with the lowest collateralization ratio will automatically be chosen.

2. Liquidating a loan (right column)
After marking a loan, it can be liquidated. A short interval of 5 minutes between marking and liquidating is required. After this, the loan can be liquidated using the marker receipt. If the initial marker doesn't use the marker receipt within 10 minutes of marking, the loan can be liquidated without owning the receipt.
