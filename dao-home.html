Stabilis DAO

You have found your way to the unofficial Stabilis DAO interface as provided to you by Stabilis Labs.

The Stabilis protocol enables existence of the stable asset STAB, and is governed by a DAO. Using this interface's pages, you can help in this endeavour:

The Incentives page is used to farm ILIS by staking resources deemed stakable by the DAO. Updating the protocol through this page also nets you some ILIS.

The Governance page is used to monitor and vote on proposals. Voting is done with staked ILIS (the DAOs governance token).

To prevent being unpleasantly surprised, it comes highly recommended to read the documentation before interacting with the interface!

If you want to interact with the Stabilis protocol directly, please visit the app interface instead.
